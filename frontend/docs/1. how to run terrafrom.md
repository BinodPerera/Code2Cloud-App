# How to Run Terraform for Frontend Deployment (GCP Cloud Run)

This guide walks through deploying the React frontend to Google Cloud Run using Terraform.

## 📋 Prerequisites

1. **Google Cloud SDK**: Installed and authenticated locally for Terraform execution.
   ```bash
   gcloud auth login
   gcloud config set project [GCP_PROJECT_ID] #optional: Select the project
   gcloud config get-value project #optional: Verify the project
   ```
2. **Docker CLI**: Installed and logged in to account with push permissions.
   ```bash
   docker login
   ```
2. **Terraform**: Installed (v1.0+ recommended).
3. **GCP Project**: A project with billing enabled.
4. **GitHub Secrets**: Ensure the following are set in your GitHub Repository Secrets:
   - `DOCKERHUB_USERNAME`: Your Docker Hub username.
   - `DOCKERHUB_TOKEN`: Your Docker Hub access token.
   - `VITE_API_BASE_URL`: The production URL of your backend API (e.g., https://api.yourdomain.com/api/v1).

---

## 🚀 Step-by-Step Deployment

### 1. Build and Push Docker Image (Automated)

Pushing changes to the `master` branch on the `frontend/**` path triggers the **CI Pipeline Frontend** workflow, which automatically builds and pushes the image to Docker Hub at `[DOCKERHUB_USERNAME]/code2cloud-frontend:latest`.

*Alternatively, to build manually:*
```bash
cd frontend
docker build \
  --platform linux/amd64 \
  --provenance=false \
  --sbom=false \
  --build-arg VITE_API_BASE_URL="https://code2cloud-be.ddns.net/api/v1" \
  -t binod07/code2cloud-frontend:v1 .
docker push binod07/code2cloud-frontend:v1
```

### 2. Initialize Terraform

Navigate into the `terraform` folder and initialize the provider plugins state.

```bash
cd frontend/terraform
terraform init
```

### 3. Deploy Resources

Run `terraform apply` passing your inputs. Include `docker.io` prefix for images.

```bash
terraform apply \
  -var="project_id=[GCP_PROJECT_ID]" \
  -var="image_uri=docker.io/[DOCKERHUB_USERNAME]/code2cloud-frontend:latest" \
  -var="region=us-central1"
```

*Replace templates with your actual configurations.*

### 4. Access Your App

Once setup completes, Terraform will output the Cloud Run URL:
```bash
Outputs:
frontend_url = "https://code2cloud-frontend-xxxxxx.a.run.app"
```
